Somehow, we must make a discrimination between these two events. By correlation coeeficients alone, it does not appear that a difference can be seen. What we can do is find the eigenvalues and eigenvectors which will discriminate between the two events.

In [156]:
from scipy import linalg
covariant_sum = np.sum([event1_cov, event2_cov], axis = 0)
eigenvalues, eigenvectors = linalg.eigh(event1_cov, covariant_sum)

These eigenvalues are then give from least to greatest. In all cases for CSP, we want to select the eigenvalues that are on the extremes. We can then sort the data by taking the absolute value of the data after subtracting 0.5.

Now we can select the eigenvectors in the order of their greatest importance. From there, we can determine the filters and the patterns resulting from CSP. After transposign the eigenvectors, the rows of the corresponging matrix are then considered as filters. When picking the number of components, we are going to pick the number of components that result in the greatest discrimination. Therefore, the first 'x' number of components will be chosen.

In [154]:
filters = eigenvectors[:,ix]
chosen_filters = eigenvectors[:4,ix]

The patterns of the data are the inverse of the eigenvectors.
